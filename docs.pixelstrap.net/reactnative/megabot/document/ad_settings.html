Ads Settings

This page contains Ads keys and id information about the theme's default settings.

1. Go to settings -> ads setting -> AdMobile Settings

Ad Publish Id: set your app Ad publish id from here.

Ad App Id: set your add app id from here.

2. Go to settings -> ads setting -> Open APP Ads

Open Ads ID: set your Open Ads ID from here.

RateApp Android Id: set your RateApp Android Id from here.

RateApp IOS Id: set your RateApp IOS Id from here.

3. Go to settings -> ads setting -> Native Ad

Native Ad set your Native ad id from here.

Ad Banner Android Id set your Ad Banner Android Id From Here.

Ad Banner Ios Id set your Ad Banner Ios Id from here.

4. Go to settings -> ads setting -> Ad Banner

Ad Banner Android Id set your Ad Banner Android Id From Here.

Ad Banner Ios Id set your Ad Banner Ios Id from here.

5. Go to settings -> ads setting -> Ad Reward

Ad Reward Android Id set your Ad Reward Android Id From Here.

Ad Reward Ios Id set your Ad Reward Ios Id from here.

Ad Interstitial Android Id set your Ad Interstitial Android Id From Here.
