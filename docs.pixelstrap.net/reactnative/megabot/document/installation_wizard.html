Installation Guide

1) Unzip & Upload To Server

Downloaded archive packages should be unzipped. Upload the ocenus directory to your web server using FTP or Control Panel after renaming it to the appropriate directory name.

Through your server's management panel, create a database for Ocenus. You may also create a database using phpMyAdmin if it is installed on your server.

Go to your browser and open the link for running project.

Open the terminal first, then hit this command to update the composer. 

composer update

2) Execute the Installation Wizard

PHP Extentions

Before moving on to the following step, confirm that the mentioned extension is installed in your device's.

3) Please make sure you have set the correct permissions for the directories listed below. click on next.

4) Please enter your Username & Purchase code below that you have already gotten.

Where is my purchase code ?

5) Configure the database and admin role here.

DB Username :- Please enter the username of phpmyadmin

DB Password :- Please enter the passwork of phpmyadmin

Database Name :- Please enter the name of the database you created before installing the project

5) Here, we finished the setup.
