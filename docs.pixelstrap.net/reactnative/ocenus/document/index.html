Server Requirement

The following characteristics must be present on your system in order for a programme to function as intended and fulfil the minimum requirements.

1. Verify your server

For Ocenus to run, you must have a configured Apache/Nginx Server for linux. If you have Windows, you should have an XAMPP server configured.

2. Verify php

Open the terminal or command prompt by running the next command on your terminal, verify the PHP version. PHP v8.1 or a more recent version is needed.

php -v

If you haven't installed it, be sure to install it first.

PHP Extensions

The following PHP extensions need to be enabled on your server.

  • PHP >= 8.1
  • BCMath PHP Extension
  • Ctype PHP Extension
  • cURL PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PCRE PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

You can check whether the above extension is installed or not, Open the terminal or command prompt and type the php -m command.
