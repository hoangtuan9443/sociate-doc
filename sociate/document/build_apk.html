How to build APK or AAB ?

  1. Open your folder in Visual studio code.
  2. Now open terminal and copy below command and paste in terminal 
    3. npm install -g eas-cli

  3. Log in to your Expo account by terminal
    4. eas login

    If you don't have an account, please register by click on link below

    Expo Sign Up
  4. Configure an Android or an iOS project for EAS Build, run the following command
    5. eas build:configure
  5. Run a build

    6. For Android: 

    eas build -p android --profile preview

    For IOS: 

    eas build --platform ios

    For both of them (Note: If you use this command, file type you will receive is only aab)

    eas build --platform all
  6. After finish all process, you wait for the build to complete and receive a link to download file
