Create Project
  1. Download zip code from Codecanyon.
  2. Now Extract the Zip.
  3. Now open this folder in Visual Studio Code and copy paste to your project
  4. Now Copy package.json folder and paste in your created project
  5. Now open terminal and copy below command and paste in terminal 
    6. npm install

  6. If you face any error while installing node modules try 
    7. npm install --legacy-peer-deps

    If above command also don't work then only try

    npm install --force
